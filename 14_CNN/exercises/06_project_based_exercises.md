# Project-Based CNN Exercises

## Project 1: Image Classification Pipeline
**Full end-to-end pipeline from data to deployment**
- Data collection and preprocessing
- Model training with monitoring
- Evaluation and validation
- Deployment as REST API

## Project 2: Real-Time Object Detector
**Live object detection from webcam**
- Implement YOLOv8 inference
- Real-time performance optimization
- Confidence filtering and NMS
- Visualization with bounding boxes

## Project 3: Medical Image Analysis
**Disease detection in medical scans**
- Fine-tune on chest X-ray dataset
- Multi-class classification
- ROC-AUC evaluation
- Clinical validation metrics

## Project 4: Scene Understanding
**Multi-label scene classification**
- Identify multiple scene categories
- Implement multi-label loss functions
- Evaluate with mAP metric

## Project 5: Image Similarity Search
**Content-based image retrieval system**
- Extract CNN embeddings
- Implement similarity matching
- Build search index
- Retrieve similar images

## Project 6: Fashion Recognition
**Clothing item classification**
- Classify clothing types
- Suggest similar products
- Implement recommendation system

## Project 7: Facial Recognition
**Face detection and identification**
- Implement face detection
- Extract facial embeddings
- Verification and identification

## Project 8: Video Action Recognition
**Classify human actions in videos**
- 3D CNN or temporal modeling
- Video preprocessing
- Action classification

## Project 9: Quality Control System
**Detect manufacturing defects**
- Anomaly detection in products
- Real-time inspection
- Defect localization

## Project 10: Document Classification
**Classify document images**
- Document type recognition
- OCR integration
- Batch processing pipeline

## Project Requirements

Each project should include:
- Data pipeline with validation
- Model training with hyperparameter tuning
- Evaluation metrics and analysis
- Performance optimization
- Documentation and reproducibility
- Test cases and edge case handling

## Evaluation Rubric

- **Functionality** (40%): Works as intended
- **Performance** (30%): Accuracy and speed
- **Code Quality** (20%): Clean, documented code
- **Innovation** (10%): Novel approaches or optimizations

## Deliverables

- Working code with version control
- Trained model checkpoint
- Training logs and metrics
- README with instructions
- Unit tests
- Performance report
