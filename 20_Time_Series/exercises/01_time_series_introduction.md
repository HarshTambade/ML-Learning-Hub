# Exercise 1: Time Series Introduction

## Problem Statement
You are given a dataset with daily stock prices. Your task is to:
1. Load and explore the dataset
2. Identify trends and seasonality
3. Calculate key statistics
4. Create visualizations

## Dataset
- Date: Daily timestamps
- Open, High, Low, Close prices
- Volume of trades
- 1000+ trading days

## Tasks

### Task 1: Data Loading and Exploration
```
- Load the CSV file
- Display first 5 rows
- Check data types
- Identify missing values
- Display summary statistics
```

### Task 2: Time Series Visualization
```
- Plot closing price over time
- Identify any obvious patterns
- Mark significant events if any
```

### Task 3: Statistical Analysis
```
- Calculate rolling mean (7-day and 30-day)
- Calculate rolling standard deviation
- Compute daily returns
- Analyze return distribution
```

### Task 4: Stationarity Testing
```
- Apply Augmented Dickey-Fuller test
- Interpret the results
- Apply differencing if non-stationary
- Retest stationarity
```

## Expected Outcomes
- Identify if data is stationary
- Understand trend components
- Recognize seasonal patterns
- Prepare data for forecasting models

## Learning Objectives
- Understand time series structure
- Learn data exploration techniques
- Apply statistical tests
- Prepare data for modeling
