# KNN Projects

## Project 1: Iris Flower Classification
**Difficulty Level**: Beginner

### Project Description:
Build a complete KNN classification system to classify iris flowers into three species (setosa, versicolor, virginica) based on their features.

### Key Components:
1. Data loading and exploration
2. Feature scaling and preprocessing
3. Train-test split
4. KNN model training with different K values
5. Model evaluation and comparison
6. Visualization of decision boundaries
7. Hyperparameter optimization

### Expected Outcome:
- Accuracy > 95%
- Clear visualization of classification results
- Analysis of optimal K value

### Datasets:
- Iris dataset (150 samples, 4 features, 3 classes)

## Project 2: Digit Recognition (MNIST)
**Difficulty Level**: Intermediate

### Project Description:
Build a KNN-based handwritten digit recognition system using the MNIST dataset.

### Key Components:
1. Load MNIST dataset
2. Data preprocessing and normalization
3. Dimensionality reduction (PCA) for efficiency
4. Train KNN model
5. Evaluate on test set
6. Analyze misclassifications
7. Performance optimization using different algorithms (brute, KD-tree, Ball-tree)

### Expected Outcome:
- Accuracy > 95% on test set
- Comparison of algorithm performance
- Analysis of computational efficiency

### Datasets:
- MNIST (70,000 images, 784 features, 10 classes)
