# Project 3: Demand Forecasting

## Objective
Forecast product demand for inventory management and planning.

## Data
- Historical sales/demand data (2+ years)
- Features: Daily demand, promotions, holidays, competitor prices
- Multiple product categories

## Tasks
1. **Data Collection**
   - Aggregate sales by product/category
   - Include external features
   - Handle missing data

2. **Exploratory Analysis**
   - Demand patterns
   - Seasonal trends
   - Promotional effects

3. **Model Development**
   - Univariate models for each product
   - Include external regressors
   - Compare ARIMA, Prophet, LSTM

4. **Business Implementation**
   - Forecast accuracy metrics
   - Inventory recommendations
   - Cost-benefit analysis

## Deliverables
- Demand forecasts for 30-90 days
- Confidence intervals
- Inventory policy recommendations
- Implementation guide

## Impact
- Reduce inventory costs
- Minimize stockouts
- Improve supply chain planning
