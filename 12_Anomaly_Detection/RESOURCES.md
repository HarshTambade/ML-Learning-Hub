## Key Concepts Quick Reference

### Performance Metrics
- **Precision-Recall Curve**: Good for imbalanced datasets
- **ROC-AUC Curve**: Overall model performance
- **F1-Score**: Balance between precision and recall
- **Contamination Rate**: Expected proportion of anomalies

### Algorithm Selection Guide
- **Fast & Scalable**: Isolation Forest, Statistical Methods
- **Density-Based**: Local Outlier Factor, DBSCAN
- **Complex Patterns**: Autoencoders, Deep Learning
- **High Dimensions**: One-Class SVM, Kernel Methods

## Getting Started Path

1. Start with foundational concepts (Z-Score, IQR)
2. Learn Isolation Forest (practical and efficient)
3. Explore LOF (density-based approach)
4. Study One-Class SVM (kernel methods)
5. Dive into deep learning approaches
6. Apply to real-world datasets
