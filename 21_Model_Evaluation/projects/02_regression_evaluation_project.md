## Objective

- Load regression dataset
- Train/test split (80/20)
- Check target distribution
- Identify outliers

### 2. Baseline Model
- Mean predictor (always predict mean)
- Establish benchmark

### 3. Model Training

- Polynomial regression
- Random forest regressor
- Gradient boosting regressor

### 4. Error Metrics

- MSE (Mean Squared Error)

- Report mean and std
- Check stability

### 8. Model Comparison
- Compare all metrics

- Trade-offs analysis

### 9. Feature Analysis

- Correlation analysis
- Impact on performance

### 10. Diagnostic Plots
- Predicted vs actual
- Residuals plot
- Learning curves
- Error distribution

## Expected Outputs
