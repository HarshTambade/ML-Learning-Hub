# RL Code Examples

Practical implementations of reinforcement learning algorithms.

## Contents

- **01_q_learning_gridworld.py**: Q-Learning agent solving a gridworld navigation task
  - GridWorld environment with obstacles
  - QLearningAgent with epsilon-greedy exploration
  - Training and testing loop
  - Performance visualization

## Getting Started

```bash
python 01_q_learning_gridworld.py
```

## Key Algorithms Implemented

- Q-Learning: Model-free RL algorithm
- Epsilon-Greedy: Exploration-exploitation balance
- Temporal Difference: Learning from bootstrapped values

## Requirements

- NumPy
- Matplotlib
