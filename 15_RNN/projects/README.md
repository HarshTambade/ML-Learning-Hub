# RNN Projects

Real-world applications that combine RNN concepts into complete systems.

## Project List

1. **01_sentiment_analysis_pipeline.py** - End-to-end sentiment analysis system
   - Data preprocessing and tokenization
   - LSTM model training
   - Inference pipeline
   - Evaluation metrics

2. **02_machine_translation.py** - Neural machine translation (English to French)
   - Encoder-decoder architecture
   - Attention mechanism implementation
   - Beam search decoding
   - BLEU score evaluation

3. **03_language_modeling.py** - Character-level language generation
   - RNN-based language model
   - Text generation
   - Perplexity calculation
   - Creative text sampling

4. **04_chatbot_seq2seq.py** - Simple conversation chatbot
   - Seq2Seq encoder-decoder
   - Intent classification
   - Response generation
   - Conversation flow management

## Project Structure

Each project includes:
- Data preparation and loading
- Model definition and training
- Inference and evaluation
- Example usage
- Performance metrics

## Getting Started

1. Choose a project aligned with your interests
2. Review the code and understand the architecture
3. Run the project with sample data
4. Experiment with hyperparameters
5. Extend with your own datasets

## Learning Outcomes

Completing these projects will help you:
- Build end-to-end ML systems
- Handle real-world data preprocessing
- Implement advanced RNN architectures
- Evaluate model performance
- Deploy models for inference
