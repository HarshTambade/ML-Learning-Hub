This folder contains 8 real-world projects to apply dimensionality reduction techniques across different domains.

- Project 1: Image Compression using PCA
- Project 3: Customer Segmentation Pipeline

**Intermediate Projects**
- Project 5: Anomaly Detection in Financial Data
- Project 7: NLP Text Classification Pipeline

**Advanced Projects**
- Project 2: Gene Expression Analysis (Bioinformatics)
- Project 4: MNIST Autoencoder for Digit Generation (Deep Learning)
- Project 6: Time Series Visualization with UMAP
- Project 8: Benchmark Comparison Tool

## Getting Started

1. Choose a project based on your interests and skill level
2. Read the project description and objectives
3. Implement the solution step by step
4. Document your findings and results
5. Submit via GitHub with proper documentation
